# aws-mfa-cli
Tiny Go program to set up a temporary AWS profile for MFA + CLI access

#### go get

```
$ go get -u github.com/Preetam/aws-mfa-cli
```

#### Binary

Binaries are available on the [Releases](https://github.com/Preetam/aws-mfa-cli/releases) page.
